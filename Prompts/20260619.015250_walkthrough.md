# USER REQUEST
Create a complete Android application in Java (NOT Kotlin) named **"Contacts SMS Call Backup Manager"** that backs up Contacts, SMS messages, and Call Logs to user-selected storage locations.

## Technical Requirements

### Platform

* Android Studio project
* Java only (no Kotlin files)
* Minimum SDK: Android 8.0 (API 26)
* Target SDK: Latest stable Android SDK
* AndroidX libraries
* Material Design UI

### App Features

#### 1. Contacts Backup

* Read all contacts from the device using ContactsContract APIs.
* Extract:

  * Display Name
  * Mobile Numbers
  * Email Addresses
  * Organization
  * Notes (if available)
* Save contacts as:

  * CSV file
  * JSON file
  * XML file
* User can choose preferred export format.

#### 2. SMS Backup

* Read all SMS messages from inbox, sent, draft, and other folders.
* Extract:

  * Sender/Receiver
  * Message Body
  * Date/Time
  * Message Type
  * Thread ID
* Save as:

  * JSON
  * XML
  * CSV
* Support large SMS databases efficiently.

#### 3. Call Log Backup

* Read call history using CallLog APIs.
* Extract:

  * Phone Number
  * Contact Name
  * Call Type (Incoming, Outgoing, Missed, Rejected, Blocked)
  * Duration
  * Date/Time
* Save as:

  * CSV
  * JSON
  * XML

### Storage & Backup Location

Implement configurable backup destination:

#### Option A - Internal Storage

* App-specific folder

#### Option B - External Storage

* User-selected folder using Storage Access Framework (SAF)

#### Option C - SD Card

* If available and permission granted

Persist selected folder URI using:

* takePersistableUriPermission()

Default folder structure:

BackupManager/
├── Contacts/
│   ├── contacts_yyyyMMdd_HHmmss.csv
│   └── contacts_yyyyMMdd_HHmmss.json
│   └── contacts_yyyyMMdd_HHmmss.xml
├── SMS/
│   ├── sms_yyyyMMdd_HHmmss.csv
│   └── sms_yyyyMMdd_HHmmss.json
│   └── sms_yyyyMMdd_HHmmss.xml
└── CallLogs/
├── calllogs_yyyyMMdd_HHmmss.csv
└── calllogs_yyyyMMdd_HHmmss.json
└── calllogs_yyyyMMdd_HHmmss.xml

### UI Requirements

Main Screen should contain:

1. Backup Destination section

   * Current selected folder
   * Change Folder button

2. Backup Cards

   * Backup Contacts
   * Backup SMS
   * Backup Call Logs
   * Backup All

3. Backup Format Selection

   * CSV

   * Any combination

4. Status Area

   * Progress Bar
   * Current Operation
   * Last Backup Time
   * Success/Error messages

5. Backup History Screen

   * List previous backup files
   * Open folder button

Request runtime permissions properly:

* READ_CONTACTS
